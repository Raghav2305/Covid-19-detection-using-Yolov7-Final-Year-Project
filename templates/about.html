About

The current COVID-19 pandemic threatens human life, health, and productivity. AI plays an essential role in COVID-19 case classification as we can apply machine learning models on COVID-19 case data to predict infectious cases and recovery rates using chest x-ray images. AI can help in the early detection of the disease, highlight the most infected areas and provide the best recovery options in a short period. In this paper, we have tried to predict the existence of the Coronavirus by performing feature detection on chest x-ray images. We collected real-life patient data from various accurate online sources and used it as input for our model. Then the chest X-Ray photographs were precisely annotated into two classes, i.e. (infected and non-infected) as required by the model. We trained the model on the collected data after conducting boundary detection using Convolutional Neural Networks (CNN) to zero in on the infected area. We then leveraged the YOLO (You only look once) object detection model on top of the CNN to train the model and classify the images as infected or non-infected. We selected our model parameters, such as max_batches and number of filters, based on the number of classes we have. After completing all iterations, we assessed our model using the MAP (mean average precision) values calculated after every 1000 iterations and tested it on unseen images by utilizing the best weights downloaded from the model. During the model training stage, we tried to identify which factors affect model prediction accuracy and loss, like activation function, model optimizer, etc. While concluding the project, we concluded that YOLO is one of the best choices for developing and training a feature detection model as it is an end-to-end object detection mechanism. It has the advantages of detection speed and accuracy and meets the real-time requirements for several use cases.
